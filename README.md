## Repository Contents

- `IPBES_WP_OpenAlex.qmd` - Quarto source document
- `bibliography.bib` - bibliography used for citations
- `figures/` - figures used in the white paper

## Build Locally

```bash
quarto render IPBES_WP_OpenAlex.qmd
```

This renders HTML, PDF, and DOCX outputs.
